프로그래밍의 역사

프로그래밍 언어의 변천사

1950년대 이전

-1940년대 말 폰 노이만에 의해 세계 최초로 프로그램 내장 방식의 컴퓨터가 등장
-일반적으로 말하는 프로그래밍 언어는 프로그램 내장 방식 컴퓨터에 대한 계산 과정을 기술 할 수 있을 때 부터 시작

1950년대

-어려운 기계어의 코드를 쉬운 명령어로 바꿔 나타내기 시작함
-A-0이라고 불리는 최초의 컴파일러 등장
-FORTRAN I 라고 하는 최초의 대중 컴퓨터 언어 등장

1960년대

-간단한 문법 구조를 가지고 문자열,수치,배열데이터 타입을 제공 해주는 "BASIC"의 등장
-아이들에게 프로그래밍과 문제 해결 원리를 가르치기위한 교육용 프로그래밍 언어 등장

1970년대

-Pascal 이라는 교육용 언어 등장,배열및 정수의 부분집합을 새로운 타입으로 정의 함
-C 의 등장
-컴파일 없이 수행되는 언어인 Awk 등장

1980년대

-C 에서 파생된 언어인 C++의 등장,클래스 상속 그리고 객체 지향이 특징이다
-높은 효율과 쉬운 사용법을 가진 Python등장

1990년대

-통합 개발 환경을 위하여 시각적 편리함을 위해 BASIC을 GUI적으로 발전 시킨 Visual Basic 등장
-Jaba의 등장, 단순하고 편리한 객체 지향언어로 처음에는 가전제품의 소프트웨어를 개발하기위해 만들어짐

2000년대

-C++과 Visual Basic의 편의성을 결합하여 마이크로 소프트에서 만든 C#이 등장
